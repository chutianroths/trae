## 选型结论
1. **满足需求覆盖**：三款模型均提供图像编辑（抠图/修复）与文生图扩散能力，并可在国内网络直连。
2. **免费额度**：皆提供可复用的官方免费额度，且配额可在控制台即时查看；前端可基于 quotaNotes 字段展示。
3. **API Key 行为一致**：三家均采用“控制台生成密钥→HTTP API 调用”方式，可统一在 .env 与 SettingsPanel 中收集。
4. **风险提示**：额度与策略会随官方活动调整，文档中需注明“以官方公告为准”并列出常见错误码。

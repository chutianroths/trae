- **输出契约**：撰写模型列表，包括名称、平台、免费额度描述、API Key 获取方式、官方链接、常见限制（实名认证、调用频率等）。
- **实现约束**：至少 3 款；引用官方文档或权威渠道；在 docs/图片编辑免费模型 下记录。
- **依赖关系**：无；后续任务依赖调研结果。
